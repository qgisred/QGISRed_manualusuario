# Creación de Elementos

Usa la barra de herramientas de QGISRed para añadir elementos a tu modelo:

![Barra de Herramientas](../assets/images/image70.png)

| Icono | Elemento | Acción |
| :--- | :--- | :--- |
| ![Pipe](assets/icons/pipe.png) | **Tubería** | Clic para inicio y fin. Crea nudos automáticamente. |
| ![Tank](assets/icons/tank.png) | **Depósito/Embalse** | Haz clic sobre un nudo existente para convertirlo. |
| ![Valve](assets/icons/valve.png) | **Válvula/Bomba** | Haz clic sobre una tubería para insertarla. |

> [!NOTE]
> Las válvulas y bombas dividen la tubería original de forma inteligente.
