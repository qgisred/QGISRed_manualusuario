# Gestión de Proyectos

QGISRed utiliza un **Gestor de Proyectos** para que puedas cambiar entre diferentes redes de forma rápida.

![Menu QGISRed](../assets/images/image14.png)

### Opciones disponibles:
*   **Crear Proyecto**: Genera la estructura de capas necesaria.
*   **Abrir Proyecto**: Carga un proyecto existente desde el disco.
*   **Importar Proyecto**: Carga datos desde un archivo comprimido o `.INP`.
*   **Clonar**: Crea una copia exacta de tu proyecto actual en una nueva carpeta.

![Dialogo Clonar](../assets/images/image22.png)
