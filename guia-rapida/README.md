# 🧭 Guía Rápida

¿Acabas de empezar con QGISRed? Aquí tienes los conceptos básicos para ponerte en marcha.

1.  **Crea un Proyecto**: Usa el [Gestor de Proyectos](archivo.md) para organizar tus archivos.
2.  **Añade Datos**: Importa desde archivos `.SHP` o `.INP`.
3.  **Dibuja tu Red**: Usa las herramientas de [Edición](../edicion/README.md).
4.  **Simula**: Obtén resultados hidráulicos en segundos.

---
> 💡 **NOTA**:
> Todos los datos se guardan en archivos locales bajo una estructura relacional compatible con EPANET.
