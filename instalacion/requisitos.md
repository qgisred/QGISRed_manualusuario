# Requisitos del sistema

Para un funcionamiento óptimo de QGISRed, asegúrate de cumplir con lo siguiente:

*   **Sistema Operativo**: Windows (x86 o x64).
*   **Software base**: QGIS versión 3.2 hasta 3.99.
*   **Componentes**: .NET Framework 4.5 instalado.
*   **Conexión a Internet**: Recomendada para la descarga inicial de dependencias.
